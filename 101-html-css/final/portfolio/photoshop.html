Week 1

Utilize Rectangular Marquee Tool, Elliptical Marquee Tool & Polygonal Lasso Tool to remove the backgrounds from the images

Week 2

Courthouse: utlize magnetic lasso tool to remove the background and add fireworks as the new background. Extra Credit: add a color overlay so the image looks more realistic. Scarecrow: utlize magic wand tool to remove the background, add it to the foreground of the house image. Extra Credit: use the rectangular marquee image to frame the new scarecrow image.

Week 3

Remove background from archway image. Duplicate and enlarge the sky image, utlize patch & spot tools to hide the edges. Add gradient overlay, poster edges filter and outer glow.

Week 4

Utlize layer mask and blend images. Use layer styles and pattern overlay, add drop shadow. Use clipping mask to create framed image. Create layer groups to organize and add type.

Week 5

Add color to a greyscale image. Extra Credit: change your own photo to greyscale and add color.

Week 6

Horse & Flowers: using adjustment layer and histogram, adjust brightness. Bluebeard: utlizing patch healing tools, clean up scratches and remove large blemish and a few other blemishes. Keith Richards: utlizing healing tools, touch up image and make one side of his face look younger. Extra Credit: add color to horse image after converting to greyscale.

Week 7

Add type. Add gradient overlay to type. Warp text. Add second line of type and copy layer styles from the first line of type. Add group folder and duplicate. On duplicated folder utlize liquify filter on text. Duplicate brick layer, add on top and utlize blend mode & opacity to alter type so it looks like it's on the wall.

Final Project

Baby: utlize healing tools to clean up face and remove the scratch from the image. Rainy Window: using layer mask and brush, blend two images together. Canal Scene: use levels adjustment to fix brightness. Flower: use magnetic lasso to remove background. Flower Blossoms: use quick selection tool to remove background. Heart and Water drop: use clipping mask to move images into shapes provided.
